# Blinkit Performance Dashboard (Power BI Project)

## Project Overview

This project presents an interactive **Power BI dashboard** built to analyze Blinkit's business performance. The dashboard provides insights into sales trends, customer behavior, and operational efficiency, enabling data-driven decision-making.

---

## Objectives

* Analyze overall sales performance
* Identify top-performing products and categories
* Understand customer purchasing patterns
* Track key performance indicators (KPIs)
* Provide actionable insights for business growth

---

## Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Excel / CSV** – Data source
* **Data Cleaning** – Power Query Editor
* **DAX** – Calculations and measures

---

## Key Features

* Interactive and dynamic dashboard
* Sales trend analysis over time
* Category-wise and product-wise performance
* Customer insights and segmentation
* KPI indicators for quick decision-making

---

## Project Files

* `Blinkit Dashboard.pbix` – Power BI dashboard file
* `Dataset` – Raw data used for analysis

---

## Insights Gained

* Identified high-revenue generating products
* Analyzed peak sales periods
* Discovered customer buying trends
* Improved understanding of business performance
